# KidSmart AI Training — Graphic Designer Brief

## The project
A fun, mobile-first learning website that teaches AI to **~12-year-olds** through a
10-step "adventure," with quizzes, a leaderboard, and play-with-friends features.
We need the **visual identity and look & feel** designed around our existing logo.

## The one rule
**The logo is the brand. Everything must be built from it.** Match the logo's style so the
whole site feels like one set — same character, colours, shapes, and personality.
*(Logo: `kidsmart_logo.png` — the soft yellow star with a graduation cap, sprout,
paintbrush, blue orbit ring, and soccer ball.)*

## Personality
**Playful but smart — warm, friendly, hand-made, encouraging.** Cool enough for a
12-year-old; never babyish, never corporate or clinical.

## Colour palette
| Colour | Hex | Use |
|---|---|---|
| Yellow | `#FFD135` | hero / mascot / primary energy |
| Blue | `#4B9FD4` | headings, accents, the "knowledge" orbit |
| Dark blue | `#2E6FA3` | headline text, borders |
| Green | `#6DBE47` | success / growth |
| Orange | `#F5A623` | highlights |
| Coral | `#E85C3A` | main call-to-action buttons / alerts |
| Off-white | `#F4F9FF` | page background |
| Cream | `#FFF8E8` | cards / panels |
| Body text | `#444444` | paragraph text |

Soft and warm (lightly pastel). Add a **lavender/periwinkle** (it's in the logo's cap +
orbit) and keep all text combinations **easy to read** (WCAG AA).

## Typography
- **Headings:** rounded, chunky, friendly — like the logo's lettering (e.g. Nunito /
  Fredoka / Baloo). Big and warm.
- **Body:** a clean, very readable sans (e.g. Inter).

## Visual style
- **Soft, rounded, organic** shapes — pillowy "squircle" cards and blobby buttons; no sharp
  corners or hard geometry.
- **Hand-drawn warmth** — gentle, slightly imperfect lines; soft, sunny shadows; a few
  tasteful doodle accents (scattered dots, a sketchy underline, the orbit swoosh).
- **One consistent icon/illustration style** drawn from the logo (rounded, soft-filled).

## The mascot ("Sunny" — the star)
Make the logo star a **character that appears throughout** the site, not just a header logo.
Design a small set of poses/expressions:
**waving hello · pointing/guiding · thinking · cheering a win · gentle "try again" ·
graduating (big win).**
Keep its symbols meaningful: **sprout = growth, cap = mastery, paintbrush = create,
orbit = curiosity, ball = play/compete** (reuse these five as motif icons too).

## Screens to style (mobile-first, 375px wide)
1. Home / welcome page
2. The 10-step **adventure map** (the signature screen — a winding journey of stops)
3. A **lesson screen** (clean, one section at a time)
4. **Quiz** + **win/celebration** screens
5. **Leaderboard** + **invite-a-friend** (link + QR) screens

## Please deliver
1. **Clean vector logo** (SVG) + **app icon / favicon set** (incl. 512px PWA icon) derived from it.
2. **Colour + type spec** (final hex values, fonts, sizes).
3. **Mascot ("Sunny") artwork set** — the poses above, as SVGs.
4. **Icon set** — UI icons + the five logo motifs, as SVGs.
5. **Mockups** of the screens above (mobile first), plus button/card/input styles.

## Must-haves
- **Mobile-first** and finger-friendly (tap targets ≥ ~44px; text ≥ 12px).
- **Readable** everywhere (AA contrast), especially soft colours on cream.
- **Child-safe & kind** — wrong-answer and empty states feel encouraging, never sad or scary.
- **Consistent** — every screen unmistakably the same brand as the logo.
