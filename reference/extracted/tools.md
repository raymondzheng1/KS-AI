# Your AI Toolkit
Before the program begins, every student and facilitator should be familiar with the tools we use throughout the two weeks. This chapter covers each tool: what it is, how to access it, and exactly how we use it in the program.
None of these tools require advanced technical skills. All are free (or have generous free tiers). Set up your accounts before Day 1 using the checklist at the end of this chapter.

## Tool 1 — Claude (Anthropic)
What It Is
Claude is an AI assistant made by Anthropic. It understands and generates human language — you talk to it in plain English (or Chinese), and it responds intelligently. Claude powers this very training program. / Access: claude.ai (web browser, no install needed) / Free tier: Available — suitable for all program activities / Used in: Days 3, 6, 7, 8, 9 (prompt engineering, creative work, project building)

### How to Use Claude in This Program
Go to claude.ai and create a free account using your email
Start a new conversation. Type your message in the input box at the bottom.
Press Enter or click the send button. Claude will respond immediately.
You can continue the conversation — Claude remembers what was said earlier in the same chat.
To start fresh (e.g. for a new activity), click "New Conversation" in the sidebar.
💡  Best Practices for Students
Always start with context: tell Claude who you are and what you're working on / Be specific: vague questions get vague answers / Iterate: if the first response isn't right, ask Claude to refine it / Verify: never accept factual claims without checking from another source / Copy useful responses into your own document — Claude conversations are temporary

## Tool 2 — Claude Code
What It Is
Claude Code is Anthropic's AI coding assistant. Unlike the web chat, Claude Code works alongside your code editor — it can read your files, write code, fix bugs, and explain concepts in the context of your actual project. / Access: Via terminal (claude command) or VS Code extension / Free tier: Available with Anthropic account / Used in: Days 8, 9 (project build days — optional for advanced students)

### Getting Started with Claude Code
📋  Setup Steps (do this before Day 8)
Option A — Web-based (easiest for beginners): / No install needed. Use claude.ai and paste your code into the chat. / Ask: "Here is my HTML file [paste code]. Can you add a button that shows the result?" / Option B — VS Code Extension (recommended for Track B projects): / Download VS Code from code.visualstudio.com (free) / Open VS Code → Click Extensions icon (left sidebar) → Search "Claude" / Install the official Anthropic Claude extension / Sign in with your Anthropic account / Highlight any code and press Cmd+Shift+P → type "Ask Claude"

📝  What Claude Code Can Do For Your Project
Write new code from a plain-English description: "Write a JavaScript function that takes a list of numbers and returns the average" / Explain code you don't understand: "Explain what this function does, line by line" / Find and fix bugs: "This code should sort alphabetically but it's not working. What's wrong?" / Suggest improvements: "How could I make this more efficient?" / Generate HTML/CSS layouts: "Create a simple webpage with a title, an image, and two buttons" / Help with your Teachable Machine export code (Track B)

⚠️  Important: Claude Code is a Collaborator, Not a Shortcut
The goal is NOT to have Claude Code write your entire project. The goal is to use it as a skilled collaborator — you make the creative and design decisions, Claude Code helps you implement them. / Always read and understand any code Claude Code produces before using it. If you don't understand it, ask Claude Code to explain it. Understanding > speed.

## Tool 3 — GitHub Copilot & Codex
What It Is
GitHub Copilot is an AI pair programmer trained on billions of lines of public code from GitHub. It suggests code completions as you type — like autocomplete but for entire functions and files. / Access: github.com/features/copilot — FREE for students via GitHub Education / Requires: GitHub account (free) + VS Code / Used in: Optional extension for Days 8–9 build projects

### Getting GitHub Copilot Free as a Student
Create a free GitHub account at github.com
Apply for the GitHub Student Developer Pack at education.github.com/pack
Verify with your school email or student ID — approval usually takes 1–3 days
Once approved, install the GitHub Copilot extension in VS Code
Sign in with your GitHub account — Copilot activates automatically
## Tool 4 — Google Teachable Machine
What It Is
Teachable Machine lets you train real AI models directly in your browser using your webcam — no code, no install. It is the primary hands-on AI tool for Days 2 and 8–9. / Access: teachablemachine.withgoogle.com / Free: Yes, completely free, no account required / Used in: Day 2 (core activity), Days 8–9 (Track B project)

### Three Types of Teachable Machine Models
Type | What It Does
Image Model | Train AI to recognise objects, gestures, or expressions from your webcam. Used in Day 2 and the Track B project.
Audio Model | Train AI to recognise sounds or spoken words from your microphone. Great for creative projects.
Pose Model | Train AI to recognise body poses and positions using your webcam. Fun for physical games and sports analysis.

## Tool 5 — Adobe Firefly
What It Is
Adobe Firefly is an AI image generator that creates images from text descriptions. Unlike some generators, it is trained only on licensed and public-domain content — making it safer for educational and commercial use. / Access: firefly.adobe.com (free tier, no age restriction) / Free: Yes — 25 generative credits per month on free tier, replenishes monthly / Used in: Day 4 (Spot the AI activity), Day 6 (AI creativity workshop)

### How to Use Adobe Firefly
Go to firefly.adobe.com
Click "Generate Image" — you can use it without signing in for basic use
Type a detailed description of the image you want (the more specific, the better)
Select a style, aspect ratio, and content type (photo, graphic, art, etc.)
Click Generate — 4 options appear. Click any to explore further.
To refine: click an image → use the editing tools on the right panel
## Tool 6 — Suno AI (Music Generation)
What It Is
Suno generates complete songs — vocals, instruments, melody — from a text description. You describe the song you want (genre, mood, theme, style) and Suno creates it in about 30 seconds. / Access: suno.com (free tier) / Free: 50 song credits per day on free tier — more than enough for all activities / Used in: Day 6 (AI creativity workshop)

## Tool 7 — Scratch + ML4Kids
What It Is
Scratch is the world's most popular visual programming platform for beginners. ML4Kids extends it with machine learning capabilities — train a model on Teachable Machine, then use it inside a Scratch project. / Scratch: scratch.mit.edu (free, no age restriction) / ML4Kids: machinelearningforkids.co.uk (free) / Used in: Optional advanced activity — combine AI + creative coding

## Tool 8 — Quick, Draw! (Google)
What It Is
Quick, Draw! is a game where you draw objects and a neural network guesses what you're drawing in real time. It is a fun, intuitive demonstration of computer vision and how AI learns from crowd-sourced data. / Access: quickdraw.withgoogle.com (free, no account needed) / Used in: Day 4 (computer vision activity)
